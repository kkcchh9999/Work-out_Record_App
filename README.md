### Work-out_Record_App

##### 2021-08-20 앱 만들기 시작  
이전에 진행하던 Memo-project 를 참고하며, 운동 일지를 기록하는 앱을 만들기로 함.  
운동 일지에는 진행할 운동 부위, 운동 루틴, 반복 횟수를 기록 가능하고 이를 데이터베이스에 저장  
운동 루틴 또한 저장하여 사용할 수 있게 만들고, 루틴의 운동 부위를 통한 검색기능과 리사이클러 뷰에서 선택 삭제 또한 구현
